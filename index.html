Multipurpose

The sulin project caters to different using areas with its docker images, iso files and containers.
Programs not included in iso and images can be added to the repository

Docker and Kubernetes:
Sulin has docker images to packaging and testing the system and to give users the opportunity to try Sulin with suitable containers.
